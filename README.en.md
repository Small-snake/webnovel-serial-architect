# Webnovel Serial Architect

[中文说明](README.md)

`webnovel-serial-architect` is a Codex skill for planning Chinese long-form serialized webnovels. It helps writers turn scattered inspiration into submission outlines, opening chapter plans, and rolling outlines that can evolve after writing begins.

## What It Helps With

- Capture raw ideas, character sparks, scene images, hooks, tropes, and market observations.
- Position a story by genre, target reader, emotional promise, selling point, and long-form viability.
- Build pitch-ready submission outlines for editors, platforms, or self-evaluation.
- Plan the first chapter, first three chapters, and first ten chapters without forcing a complete full-book outline.
- Continue after drafted chapters by extracting current story state and planning the next 5-10 chapters.
- Diagnose writer's block caused by weak goals, weak conflict, delayed payoff, setup overload, continuity issues, or too many open loops.

## Core Principle

```text
Capture inspiration first.
Build a minimum viable plan.
Start writing.
Update the outline as the serial grows.
```

The skill treats long webnovel planning as a living system. It separates decisions into fixed, tentative, and deferred items so writers can move forward without locking the entire book too early.
